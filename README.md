# React App – Search Movies (OMDb)

A React + TypeScript web app that searches movies using the **OMDb API** and displays a dedicated details page for each movie.

## Live Demo
- Netlify: https://searchamovieee.netlify.app/

## GitHub Repo
- https://github.com/sharjeel1992/React-App-Search-Movies

---

## Features (Meets Assignment Requirements)
- ✅ **React functional components** only
- ✅ **Controlled input** (movie search textbox)
- ✅ **Fetch API + useEffect** (data fetching is triggered by user search)
- ✅ **Dynamic rendering** of API results (movie grid)
- ✅ **Routing to entity details page** (`/movie/:imdbID`) showing one movie’s information
- ✅ **Reusable custom hook** (`useFetch`) for API calls
- ✅ **Responsive layout** (CSS grid adapts to screen size)
- ✅ **Search history** saved in `localStorage`
